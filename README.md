# SWOOOP Discord Bot

### Important /!\
Before you use the bot, you have to adjust the Token and id's to yours, can be found under Data
(/mySWOOOP_DC/src/main/java/de/deanomus/dc/storage/Data.java)


#### commands:

 - Administration
    - stop
 - Moderation:
    - Broadcast
    - Changename
    - Clear
    - Kick
    - Say
 - User:
    - Info
    - Ping
